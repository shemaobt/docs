# Glossary

## RFC

Request for Comments document used to evaluate decisions and capture trade-offs.

## System page

Evergreen documentation page describing current architecture and behavior for one product/system.

## Oral-first

A design principle prioritizing spoken language interactions and speech-native workflows.

## Acousteme

Project-specific representation unit used for speech-centric processing and translation experiments.
